## Пример пользовательского сценария

1. юзер открывает сайт
2. нажимает "оставить заявку"
3. заполняет форму: "Кот Барсик, 3 года, с 10 по 15 июня".
4. отправляет заявку.
5. идет в `/profile`, видит заявку со статусом `in_review`.
6. позже заходит снова - статус уже `approved`.

---

## Структура проекта

- `app/` - маршруты App Router
- `src/app/home/` - код главной
- `src/app/profile/` - код профиля
- `src/styles/` - стили

---

## Запуск проекта

установка зависимостей:
- `npm install`

запуск в дев-режиме:
- `npm run dev`

открыть:
- `http://localhost:3000`
